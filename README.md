Netgen OpenWeatherMap Bundle is an eZ Publish / eZ Platform bundle for working with http://openweathermap.org/ API.

Implemented features
--------------------

* Current weather data
* 5 day / 3 hour forecast
* 16 day / daily forecast
* UV Index
* Weather stations
* Air pollution
